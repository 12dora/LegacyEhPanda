|---|---|
| Untrusted tag markdown -> MarkdownUtil/TagTranslation | Community tag metadata can contain links/images and must not broaden URL handling without fixtures. |
| User session cookies -> DF request rewrite | Cookies must stay tied to the original request URL when DF rewrites the network host. |
| SwiftPM manifest -> dependency resolver | Manifest edits determine which code is compiled into the app. |
