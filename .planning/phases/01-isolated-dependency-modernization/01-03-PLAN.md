## Trust Boundaries

| Boundary | Description |
|---|---|
| Untrusted/cached gallery image -> color extraction | Image pixels influence UI colors and readability. |
| External package code -> app-owned module | Dominant-color algorithm becomes first-party code. |
| SwiftPM manifest -> dependency resolver | Removing a package changes compiled dependencies. |

| Threat ID | Category | Component | Severity | Disposition | Mitigation Plan |

| T-01-03-02 | Denial of Service | Image renderer/color sampling | low | mitigate | Keep generated fixture coverage for small and transparent images where practical. |
| T-01-03-SC | Tampering | SwiftPM package resolution | high | mitigate | Remove the external package only after local tests pass; do not add unaudited image packages. |
