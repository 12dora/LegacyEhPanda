typed-throws `response()` bodies for the 4 routine requests — then flip the 04-03 routine
baseline's acquisition from the Combine facade to the async path. The frozen assertions passing

is the deliberate mechanism that keeps every commit compiling — the phase ends Combine-free,
enforced by 04-13's grep gates.

participates in structured cancellation (a cancelled effect now aborts the in-flight HTTP call
instead of letting it complete unobserved). User-visible behavior is identical because TCA
discards sends from cancelled effects. Do NOT "fix" this with detached tasks.

Output: fetch helper + 4 proven async bodies.
