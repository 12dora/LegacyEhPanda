<objective>
Prepare the Combine request layer for the typed-throws async migration (CONC-01) with two
zero-behavior-change refactors:

1. Rename the Result facade to `legacyResponse()` — freeing the `response()` name. This is
   load-bearing for D-01/D-02: a compile probe on the pinned Swift 6.3.3 toolchain proved that a
   concrete `func response() async throws(AppError) -> Response` on a struct SHADOWS the
   protocol-extension facade of the same name, breaking every unmigrated call site with
   "call can throw" / type-mismatch errors. Renaming the facade first lets async bodies land
   family-by-family under the final name with a green build at every commit.
2. Add the injectable `urlSession` seam (D-07) to every request that hard-codes the shared
   session, so the Wave-0 baseline harness can substitute a counting URLProtocol stub session
   per request and execute fully offline.

Purpose: unblock incremental migration + offline parity testing without any behavior change.
Output: renamed facade + 65 renamed call sites; injected session seam on all 44 request structs.
</objective>
